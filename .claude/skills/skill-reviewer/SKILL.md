---
name: skill-reviewer
description: Looks at a given claude skill file for: proper markdown formating, clarity, and usefulness in relation to the projects structure and goals. All links in the skill are also checked to make sure they are valid and working, and if there is an associated examples.md file then this is also reviewed. Small style/formatting issues as well as typos can be automatically fixed. Larger content additions, updates, or deletions should be provided as a list of any suggested improvements.

---

# Steps

1. **Check Proper File Format**
    - Identify and fix any markdown style/formatting issues
    - Identity and fix any typos or minor grammar errors
    - Add larger improvements, updates, and/or deletions to "suggested improvements" list
2. **Validate Links**
    - All Links located in the Additional Resources section and elsewhere in the files
    - Any invalid links should be added to the "suggested improvements" list
3. **Review Assocated examples.md file (if applicable)**
    - Identify and fix any markdown style/formatting issues
    - Identity and fix any typos or minor grammar errors
    - Add larger improvements, updates, and/or deletions to "suggested improvements" list
4. **Generate Suggested Improvements**
    - Display suggested improvements identified in the previous steps
